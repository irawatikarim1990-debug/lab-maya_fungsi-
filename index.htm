Karakteristik Fungsi dalam Matematika

Memahami bentuk umum dan karakteristik grafik dari fungsi linear, fungsi kuadrat, dan fungsi eksponensial sebagai tiga keluarga fungsi fundamental yang sering digunakan dalam memodelkan fenomena dunia nyata.

Fungsi Linear

Garis Lurus
f(x) = ax + b

a = gradien (kemiringan garis)

b = perpotongan dengan sumbu Y

a > 0: garis naik · a < 0: garis turun

Domain: ℝ · Range: ℝ

Fungsi Kuadrat

Parabola
f(x) = ax² + bx + c

a = arah pembukaan parabola

Titik puncak: (-b/2a, f(-b/2a))

Sumbu simetri: x = -b/2a

Diskriminan D = b² - 4ac

Fungsi Eksponensial

Pertumbuhan
f(x) = a · bˣ

a = nilai awal (koefisien)

b = basis (b > 0, b ≠ 1)

b > 1: pertumbuhan · 0<b<1: peluruhan

Asimtot datar: y = 0

Panduan Penggunaan

Prosedur Penggunaan Lab Maya

1

Pilih Tab Simulasi

Klik tab "Simulasi" pada navigasi atas untuk memulai eksplorasi grafik fungsi secara interaktif.

2

Aktifkan Fungsi yang Dipelajari

Gunakan tombol toggle berwarna untuk mengaktifkan fungsi linear (cyan), kuadrat (kuning), atau eksponensial (merah). Bisa satu atau beberapa sekaligus.

3

Atur Nilai Parameter

Geser slider pada setiap panel fungsi untuk mengubah nilai parameter (a, b, c). Amati perubahan grafik dan persamaan secara real-time.

4

Amati Titik Penting

Perhatikan penanda titik potong, titik puncak (vertex), dan asimtot yang muncul otomatis pada grafik. Arahkan kursor ke canvas untuk melihat koordinat.

5

Analisis Tabel Nilai

Gunakan tabel nilai di bawah canvas untuk membandingkan output ketiga fungsi pada nilai x yang sama.

6

Kerjakan LKPD

Buka tab "LKPD", kerjakan kegiatan yang tersedia dengan menggunakan simulasi sebagai alat bantu. Centang setiap kegiatan yang selesai.

7

Reset dan Eksplorasi Ulang

Gunakan tombol "Reset" untuk mengembalikan semua parameter ke nilai awal. Eksplorasi dengan kombinasi parameter lainnya.

Tujuan Pembelajaran

  • Mengidentifikasi bentuk umum fungsi linear, kuadrat, dan eksponensial

Kegiatan 1: Fungsi Linear

f(x) = ax + b

Petunjuk: Buka tab Simulasi, aktifkan fungsi linear saja. Atur parameter dan amati grafik yang terbentuk.

1. Atur slider a = 2 dan b = 1. Tuliskan persamaan fungsi linear yang terbentuk!

2. Ubah nilai a menjadi negatif. Apa yang terjadi pada arah garis?

3. Tentukan titik potong grafik dengan sumbu Y ketika a = 3 dan b = -2!

4. Apa yang terjadi pada grafik jika nilai a = 0? Jelaskan!

5. Tentukan titik potong grafik f(x) = 2x - 4 dengan sumbu X!

Saya telah menyelesaikan Kegiatan 1

Kegiatan 2: Fungsi Kuadrat

f(x) = ax² + bx + c

Petunjuk: Nonaktifkan fungsi linear, aktifkan fungsi kuadrat. Atur parameter dan amati parabola yang terbentuk.

1. Atur a = 1, b = -2, c = -3. Tentukan koordinat titik puncak (vertex) parabola!

2. Tentukan persamaan sumbu simetri dari parabola pada soal no. 1!

3. Tentukan titik potong parabola dengan sumbu X (gunakan rumus diskriminan)!

4. Ubah nilai a menjadi negatif. Bagaimana perubahan bentuk parabola?

5. Apa pengaruh nilai |a| yang semakin besar terhadap lebar parabola?

6. Hitung nilai diskriminan D = b² - 4ac untuk a=1, b=-2, c=-3. Apa maknanya?

Saya telah menyelesaikan Kegiatan 2

Kegiatan 3: Fungsi Eksponensial

f(x) = a · bˣ

Petunjuk: Nonaktifkan fungsi kuadrat, aktifkan fungsi eksponensial. Atur parameter dan amati kurva yang terbentuk.

1. Atur a = 1 dan b = 2. Tentukan nilai f(3)!

2. Tentukan domain dan range dari fungsi f(x) = 2ˣ!

3. Apa yang terjadi pada grafik ketika b > 1? Jelaskan!

4. Ubah b = 0.5. Apa yang terjadi? Jelaskan konsep peluruhan!

5. Tentukan asimtot datar dari fungsi eksponensial f(x) = a · bˣ!

6. Berikan satu contoh penerapan fungsi eksponensial dalam kehidupan sehari-hari!

Saya telah menyelesaikan Kegiatan 3

Tentang Lab Maya

Lab Maya Fungsi adalah media pembelajaran interaktif berbasis web yang dikembangkan untuk membantu peserta didik memahami karakteristik dasar fungsi linear, kuadrat, dan eksponensial beserta grafiknya.
